# Risk measures

## Data

Profit/loss data
def PL = P_{t} + D(t) - P(t-1)

Present value for profit/loss data
def PLP = (P(t) + D(t))/(1+d) - P(t-1)

Forward value for profit/loss data
def PLF = P(t) + D(t) - (1 + d)P(t-1)
